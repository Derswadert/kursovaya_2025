## kursovaya_2025

### Цель проекта

Данный проект посвящён статистическому анализу гендерных различий в оплате труда в разных странах мира.

### Задачи, выполняемые проектом

В рамках работы проведён анализ данных о среднемесячных зарплатах мужчин и женщин с использованием методов математической статистики, включая:
- описательный анализ данных
- оценку параметров распределений
- проверку статистических гипотез
- идентификацию распределений

### Источник данных

https://ilostat.ilo.org/data/

International Labour Organization (ILO)
Labour force by sex and age

В датасете представлены:
- `Country` — страна
- `Sex` — пол (male/female)
- `Year` — год
- `Value` — среднемесячная зарплата (в долларах USD)

### Технологии
Python (pandas, NumPy, matplotlib, scipy)
